# INDIGINEER
INDIGNEER is an Aniwatch-style anime discovery and review website where users can post completely anonymous reviews about any anime.
No login. No identity. Just honest opinions.

This project uses a Flask backend and a single-file frontend SPA powered by the Jikan (MyAnimeList) API.
